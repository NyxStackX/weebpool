# Instructions pour créer votre CV PDF

## Option 1: Créer avec un éditeur en ligne (Recommandé)

### Canva (Gratuit)
1. Allez sur [canva.com](https://www.canva.com)
2. Cherchez "CV" dans les templates
3. Choisissez un template moderne et professionnel
4. Personnalisez avec vos informations:
   - **Informations personnelles**: Paulina Aleba, alebapaulina89@gmail.com
   - **Profil**: Développeuse Web passionnée par la création d'expériences digitales
   - **Compétences techniques**: HTML5, CSS3, JavaScript, React, Vue.js, Node.js, MongoDB, Firebase, Docker, Git
   - **Expérience**: Listez vos projets principaux
   - **Formation**: Votre formation RNCP
   - **Langues**: Français, Anglais, etc.
5. Téléchargez en PDF
6. Renommez le fichier en `CV_Paulina_Aleba.pdf`
7. Placez-le dans ce dossier (`php/assets/`)

### Resume.io (Gratuit)
1. Allez sur [resume.io](https://resume.io)
2. Créez un compte gratuit
3. Choisissez un template ATS-friendly
4. Remplissez vos informations
5. Téléchargez en PDF

## Option 2: Créer avec LaTeX (Pour les plus techniques)

Si vous maîtrisez LaTeX, vous pouvez utiliser Overleaf avec un template moderne comme:
- ModernCV
- AltaCV
- Awesome-CV

## Option 3: Créer avec Microsoft Word / Google Docs

1. Utilisez un template de CV moderne
2. Exportez en PDF
3. Assurez-vous que le fichier est optimisé (< 2MB)

## Conseils importants

1. **Gardez-le concis**: 1-2 pages maximum
2. **Utilisez des verbes d'action**: Développé, Conçu, Implémenté, Optimisé
3. **Quantifiez vos réalisations**: "Amélioration de 30% des performances"
4. **Adaptez-le**: Personnalisez selon le poste visé
5. **Vérifiez l'orthographe**: Aucune faute tolérée
6. **Format PDF**: Toujours envoyer en PDF pour préserver la mise en forme
7. **Nom du fichier**: `CV_Prenom_Nom.pdf` (professionnel)
